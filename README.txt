Commonly used options missing from Wordpress

== Description ==

These are options we all wish were built into WordPress but are not.  They need to be added with a separate plugin or code.


== Installation ==

1. Upload `extra-options.zip` to WordPress with the plugin installer.
1. Activate the plugin through the 'Plugins' menu in WordPress
